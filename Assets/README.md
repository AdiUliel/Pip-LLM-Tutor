# Assets

Project media and 3D-printing files.

| File | Description |
|---|---|
| `pip_poster.pdf` | Project poster (original, print quality) |
| `pip_poster.png` | Poster rendered as an image (embedded in the root README) |
| `3D/pip_main_body.stl` | 3D-printed enclosure — main body (holds the LCDWIKI ESP32-S3 board, speaker and button) |
| `3D/pip_rear_cover.stl` | 3D-printed enclosure — rear cover |

## 3D printing notes

The enclosure was printed in PLA with standard settings (0.2 mm layer height, no
supports needed for the rear cover). The main body is sized for the LCDWIKI 2.8"
ESP32-S3 display board with the push-to-talk button wired to the IO2/IO3
expansion header.
